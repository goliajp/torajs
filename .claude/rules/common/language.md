# Language

**ABSOLUTE RULE: Only use Chinese (简体中文) and English. Any other language (Korean, Japanese, French, etc.) is strictly forbidden in all responses, comments, and output. No exceptions.**

Always respond in Chinese (简体中文) by default. Use English only for:
- Code, variable names, and technical identifiers
- Git commit messages
- CLI command output
- Content that must remain in English (e.g., PR descriptions on English-language repos)

Using any language other than Chinese or English is a critical error that must never occur.
