# FizzBuzz — torajs example

Classic FizzBuzz, prints numbers 1..20 with `Fizz` / `Buzz` / `FizzBuzz`
substitutions on multiples of 3, 5, and 15.

## Running

```sh
tr run fizz-buzz.ts
# or compile to a native binary
tr build fizz-buzz.ts -o fizz-buzz && ./fizz-buzz
```

## Exercises

- Number-to-string conversion via `i.toString()`
- Integer modulo (`%`)
- String concatenation
- Conditional control flow (`if` / `else if` / `else`)

Output matches `bun run fizz-buzz.ts` line-for-line.
