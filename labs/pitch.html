torajs is an ahead-of-time TypeScript runtime that emits standalone native binaries. Same TS programs as Bun. No JIT, no interpreter, no embedded runtime. The result is a 37 KB executable that beats bun build --compile on every benchmark we run.

01Why this isn't another bunthree pillars
AOT TO NATIVE

Zero JITs, zero warmup

TS → LLVM IR → machine code → binary. First nanosecond runs at full throughput. Same speed at runtime t=0 and t=∞.

SMALL ARTIFACT

37 KB per executable

The runtime is the program. Ships on edge functions, IoT firmware, CLI tools, containers. No 60 MB tax per binary.

COLD-START SPEED

1.37 ms hello-world

Lambda cold-starts, container init, on-demand workloads. 15× faster than the next-best AOT TS runtime.
